🤖 MEnvAgent

Scalable Polyglot Environment Construction for Verifiable Software Engineering

An automated framework for building executable environments across 10 programming languages to enable scalable generation of verifiable software engineering data.

🖼️ Main Architecture Diagram

位置: Architecture Overview 部分

需要: MEnvAgent-main.png (1200x600px)

内容: 规划-执行-验证循环 + 环境复用机制

📝 Planning Stage

  • Repository Analysis Agent: Explores repository structure
  • Environment Setup Agent: Determines base image and installation scripts
  • Test Configuration Agent: Synthesizes test configuration

⚡ Execution Stage

  • Environment Execution Agent: Instantiates containers with real-time monitoring
  • Dynamic error resolution and automatic retry

✅ Verification Stage

  • Verification Agent: Executes tests and performs error attribution
  • Iterative refinement loop with diagnostic feedback

♻️ Environment Reuse Mechanism

  1. Retrieves similar historical environments from Environment Pool
  2. Uses EnvPatchAgent to generate incremental patches
  3. Applies patches to adapt environment to target requirements
  4. Significantly reduces computational overhead
